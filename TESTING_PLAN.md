# План ручного тестирования проекта Blueberries

## Подготовка к тестированию

### 1. Запуск серверов

**Backend:**
```bash
cd backend
npm install
npm run dev
```
- Проверить, что сервер запустился на `http://localhost:3000`
- Проверить сообщения в консоли об успешной инициализации БД и миграциях

**Frontend:**
```bash
cd frontend
npm install
npm run dev
```
- Проверить, что приложение запустилось на `http://localhost:5173`
- Проверить отсутствие ошибок в консоли браузера

## 2. Тестирование авторизации

### 2.1 Регистрация

## Примечания

- Все найденные баги должны быть задокументированы с описанием шагов для воспроизведения
- При тестировании на мобильных устройствах использовать реальные устройства или эмуляторы
- Проверять работу в разных разрешениях экрана
- Проверять работу с разными данными (пустые поля, длинные тексты и т.д.)
